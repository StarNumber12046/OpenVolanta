# OpenVolanta XPlane UDP bridge

## Why use this instead of the XPlane plugin?

This doesn't require any modifications to X-Plane, and can be used with any X-Plane version.

Bonus: It can't crash your simulator

## Features

- [x] Flight info

- [x] Flight path

- [x] Aircraft info (better than the stock plugin)

- [ ] Landings (can be done with the OpenVolanta LandingRate plugin)

- [ ] Coins all over the world (who even uses them lol)

## Installation

1. Download the latest release from the [releases](https://github.com/StarNumber12046/OpenVolanta/releases) page
2. Run the XPlane_udp.exe file
3. Enjoy :D
